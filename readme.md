특징
1. 본인의 컴퓨터에 원고가 저장되고 외부로 글이 유출되지 않습니다. 데이터베이스를 사용하지 않고 웹브라우저 주소표시줄에 데이터를 저장하는 방식입니다. 원고지에 글자를 치면 주소표시줄에 글자들이 추가되는 것을 보실 수 있습니다.
2. 새로고침해도 주소표시줄은 변하지 않으므로 데이터가 그대로 있습니다. 쓰던 글이 새로고침돼서 지워졌다는 변명이 통하지 않습니다.
3. 웹브라우저에 북마크하면 컴퓨터에 저장하는 것과 마찬가지가 됩니다. 주소표시줄의 데이터를 불러오게 되기 때문입니다.
4. 학생들이 글을 다 쓰고나면 주소표시줄을 복사해서 학급게시판에 올리거나 선생님에게 링크를 보내면, 선생님은 그 링크를 클릭하여 학생들의 원고를 확인하실 수 있습니다.
5. 단일 html 파일로도 작동합니다. 첨부된 html 파일 받아서 웹브라우저에서 실행하셔도 원고지 기능은 같습니다. 간단히 컴퓨터로 원고지 쓰는 법을 알려주고 싶으시다면 파일만 받아서 실행하셔도 좋을 것 같습니다.

메뉴 설명
1. [새 문서]를 클릭하면 써놓은 글이 다 지워진다는 경고가 나오고, 확인을 누르면 새 원고지가 나옵니다.
2. 글이 길어지면 자동으로 2쪽이 나타납니다.
3. 글꼴은 명조와 고딕 중에서 고르실 수 있습니다.
4. 원고지 색은 빨강, 회색, 검정 중에서 고르실 수 있습니다.
5. 화면 확대/축소 기능으로 학생들에게 보여주기가 쉽습니다.
6. [인쇄]를 클릭하고, 프린터를 PDF저장으로 설정하면 pdf로 저장하실 수 있습니다. 애플 기기에서는 인쇄 창에서 [공유]버튼을 누르고 파일 앱에서 저장할 위치를 지정하시면 됩니다.
7. [현재 문서 주소 복사]를 눌러 학급게시판에 올리면 선생님이나 다른 친구들이 그 링크를 열고 원고를 확인할 수 있습니다.
 
 (중요) 링크를 공유할 수 있는 많은 게시판이나 프로그램에서 (패들렛, 띵커벨보드, 카톡 등) 링크 글자수에 제한을 두고 있습니다. 글이 길어지면 원고지의 주소도 길어지고 각 게시판의 링크 글자수 제한을 넘기는 경우가 많아 올바르게 원고가 공유되지 않습니다. 이 경우 링크가 잘리기 전까지의 글자만 제대로 보이며, 화면 상단에 글이 제대로 공유되지 않았다는 안내문이 나타납니다. 
 
8. [텍스트만 복사]는 말 그대로 텍스트만 클립보드에 복사합니다. 메모장에 붙여넣기 하셔서 글을 보시면 되겠습니다.
9. 태블릿이나 핸드폰처럼 화면 너비가 좁으면 상단 햄버거 메뉴에 버튼들이 들어갑니다.
10. ctrl+z로 되돌리기, ctrl+shift+z 또는 ctrl+y로 다시실행 할 수 있습니다. 모바일을 위해서 버튼이 추가되었습니다.
